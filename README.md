# DAGGAU AI — интеллектуальный чат-ассистент для студентов и абитуриентов

DAGGAU AI — это мобильный/десктопный/веб чат-бот, созданный для помощи абитуриентам и студентам Дагестанского государственного аграрного университета (ДагГАУ).

Проект решает проблему:
информация о поступлении, документах, факультетах и обучении разбросана по сайту, и студентам сложно быстро получить точный ответ. Этот бот объединяет всё в одном диалоге.

---

## Что умеет проект (сейчас)

- Диалоговый чат с ИИ
- Контекстная память переписки
- Системный промпт, задающий поведение ассистента
- Поддержка Markdown (ответы выглядят структурированно)
- Streaming-ответы (бот «печатает» ответ в реальном времени)
- Архитектура, готовая к подключению RAG (документы, сайт университета)

---

## Стек

**Frontend**
- Python
- Flet (мобильный и десктопный UI)

**Backend / AI**
- OpenAI SDK (через OpenRouter)
- Chat Completions API
- YAML-based system prompt
- Подготовка под RAG
